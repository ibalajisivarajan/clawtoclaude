How it works
How it works 🌴
1
Find a Skill on ClawHub
Browse clawhub.ai, find a skill you want, and download the .zip. Then drop it here — everything converts in your browser, no server needed.
📦
2
Strip the Clutter
ClawHub's metadata: {clawdbot:...} and _meta.json are surgically removed. Your SKILL.md body and any extra files are kept exactly as-is.
🧹
3
Claude Polishes It
Hit ✦ Enhance and Claude AI rewrites the description so it triggers correctly in Claude.ai. Compatibility warnings flag CLI tools or env vars you'll need.
✨
4
Drop it into Claude Code
Place the .skill file in your Claude Code workspace skills/ folder — it's live instantly. For instruction-only skills (no CLI tools), you can also upload to Claude.ai Settings → Skills. 🤙
