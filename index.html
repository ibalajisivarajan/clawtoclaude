How it works
Four Easy Steps 🌴
1
Grab the Zip
Download any .skill from clawhub.ai and drop it here — or use the Search tab to find and fetch skills directly. No server needed, everything runs in your browser.
📦
2
Strip the Clutter
ClawHub's metadata: {clawdbot:...} and _meta.json are surgically removed. Your SKILL.md body and any extra files are kept exactly as-is.
🧹
3
Claude Polishes It
Hit ✦ Enhance and Claude AI rewrites the description so it triggers correctly in Claude.ai. Compatibility warnings flag CLI tools or env vars you'll need.
✨
4
Install & Surf
Download your fresh .skill file → Claude.ai → Settings → Skills → Upload. Done! Claude automatically loads it whenever it's relevant. 🤙
